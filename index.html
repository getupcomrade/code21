Школа программирования Code21

Школа программирования, в которой каждый сможет обрести навыки в IT сфере от лучших специалистов Казахстана.

О НАС

Школа программирования code21 — подразделение отечественной IT-компании 21Tech. Программы обучения рассчитаны как для новичков, так и для продвинутых IT-специалистов. Главное преимущество школы code21 в том, что вовремя обучения у учеников есть возможность закрепить знания на реальных IT-проектах и трудоустроиться в лучшие компании Казахстана, так как у школы имеются договора с компаниями по дальнейшему трудоустройству. Преподаватели школы code21 имеют опыт работы в передовых американских, европейских и казахстанских компаниях. Обучение проходит в офлайн формате в современном офисе на территории EXPO. Для закрепления знаний используется собственная LMS-платформа.

ПРОГРАММЫ ОБУЧЕНИЯ

Front -End разработка JavaScript React ES6+

Студенты будут решать проблемы, используя такие инструменты, как HTML, CSS и JavaScript:
-Создавать, отлаживать и поддерживать профессиональные веб-сайты;
-Эффективно взаимодействовать, сотрудничать и общаться с преподавателем и сокурсниками

Back-End разработка Java and Spring Boot

-Разработка серверных и веб-приложений с использованием инструментов и фреймворков разработки, таких как SpringMVC/Spring Boot и Hibernate:
- Создание и поддержание динамических веб-приложений с использованием технологий на базе Java.
- Управлять и представлять информацию для поддержки принятия важных для бизнеса решений.
- Эффективно взаимодействовать, сотрудничать и общаться с преподавателем и студентами.

Data Аналитик
Python

- Ознакомление с данными и способами их сбора, манипулирования, очистки;
- Изучение основных библиотек Python для анализа данных: NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn;
- Изучение различных статистических графиков и их использование в EDA.
